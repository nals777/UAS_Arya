JAWABAN NO 1

Cara kerja website pada dasarnya melibatkan web server, domain, dan hosting yang saling terhubung:
1. Web Server Web server merupakan software dan hardware yang bertugas menerima request dari client (browser) melalui protocol HTTP/HTTPS. Web server tempat menyimpan files dan resources yang menyusun sebuah website seperti HTML, CSS, Gambar, Javascript dan lainnya. Beberapa web server populer adalah Apache, Nginx, IIS.
2. Domain Domain merupakan alamat website yang diberikan ke komputer client untuk mengakses web server. Domain berupa nama unik yang mudah diingat, contohnya google.com. Domain mengarah ke alamat IP web server tempat website disimpan.
3. Hosting Hosting merupakan jasa layanan untuk menyewa/menyediakan server dan tempat untuk menaruh file website agar bisa diakses melalui internet.
Jadi hosting berfungsi untuk menempatkan web server beserta file website agar bisa diakses lewat domain. Jadi ketiganya saling terkait: - Website diletakkan di web server dan disimpan di hosting - Domain mengarah ke alamat IP hosting - Client mengakses domain, request diarahkan ke web server lewat hosting - Web server merespon request dan menampilkan website Dengan demikian client bisa mengakses konten website melalui domain tertentu yang di-host-kan di server hosting.
